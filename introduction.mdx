## What is Janek Unon?

Janek Unon provides you with a centralized space to create, organize, and publish your project's documentation. You get powerful MDX editing, version control integration, and customizable themes to build professional docs sites. Start by connecting your repository, and Janek Unon handles hosting, search, and navigation automatically.

<Callout kind="info" collapsed="false">
  Janek Unon supports MDX v3 and integrates seamlessly with GitHub for real-time previews.
</Callout>

## Quick Start Steps

Follow these steps to set up your first documentation site.

<Steps>
  <Step title="Connect Repository" icon="git-branch" title-type="p">
    Link your GitHub repo in the dashboard. Janek Unon pulls your `docs/` folder automatically.
  </Step>

  <Step title="Customize Config" icon="settings" title-type="p">
    Edit `janek.config.js` to set your brand color and navigation.

    ```javascript
    // janek.config.js
    export default {
      theme: {
        primaryColor: '#3B82F6',
      },
      navigation: [
        { title: 'Introduction', href: '/introduction' },
        { title: 'API Reference', href: '/api' },
      ],
    };
    ```
  </Step>

  <Step title="Publish" icon="globe" title-type="p">
    Push changes to trigger a build. Your site deploys at `your-project.janek.app`.
  </Step>

## Brand Guidelines and Customization

Follow these guidelines to align your docs with Janek Unon's branding.

<Expandable title="Advanced Customization Examples" default-open="false">
  Override default styles in your config.

  <CodeGroup show-lines="true" tabs={[]}>
    ```css
    /* styles/global.css */
    :root {
      --primary: #3B82F6;
    }
    .docs-heading {
      color: var(--primary);
    }
    ```

    ```javascript
    // tailwind.config.js
    module.exports = {
      theme: {
        extend: {
          colors: {
            primary: '#3B82F6',
          },
        },
      },
    };
    ```
  </CodeGroup>
</Expandable>

You now have everything to build stunning documentation. Customize freely while maintaining consistency with the `#3B82F6` brand color for buttons and accents.
